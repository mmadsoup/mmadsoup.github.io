Spline Editor ━ Unity / C#

Path following in a 3D space! A collab tool.


My Contributions

  • Hybrid Spline Engine - Create simple interpolated curves
  • Surface-based Point Placement - Place points using raycasting and fallback mechanism (snaps point to (0, 0, 0))
  • Dotted Line Preview - Preview how the next point affects the curve
  • Point Manipulation - Add/delete points with mouse and hotkeys
  • Inspector Customization - Readable interface for designers
  • Dynamic Path Tracking - Continuous object movement along the spline

Scripts

  • Spline
  • Spline Editor
  • Spline Menu Item

Challenge

At first I thought to use bezier curves for this tool. But after some research I found that bezier curve "handles" made creating splines more complicated to use and implement.

Solution

I pieced together a system that uses a combo of Catmull-Rom and Hermite spline calculations. With this, curves are created just by control point position, rather than handle angle or length

Collaborative Achievement

My teammate Andy expanded upon this system with advanced features!

His contributions included branching spline paths, object rotation, and movement through keyboard input and rigidbody forces.

I'm massively proud about our work together, and I definitely learned a lot from working wth him!

YarnSpinner Dialogue Editor ━ Unity / C#

A node-based dialogue graph extension for YarnSpinner!

What motivated me to build this tool was: Dialogue... Written in a code editor? It's too teduious to switch between and hard to keep track of branching paths in a growing project. I also wanted the writers on our team to get to writing without having to download and learn how to use VSCode!

Features Implemented

  • Unique Identifiers - Auto assign distinct IDs to dialogue elements (files, nodes, and groups)
  • Choice Creation - Create branching nodes by typing "->" in a text field
  • Port Drag Node Creation - Create new node by dragging on a port to create a new connected node
  • Action Nodes - Envoke yarn-specific code with yarn commands
  • Action Creation - Create an action node by typing "///" within a dialogue's text field
  • Export to Yarn File - Export the graph into a properly formatted yarn file
  • Batch Yarn File Export - Create multiple dialogues in separate yarn files
  • Graph Settings - Customize colors and dialogue graph syntax
  • Node Saving and Loading - Scriptable objects to serialize and deserialize nodes, groups and connections
  • Dialogue Nodes - Multi-line text fields for freeform writing

Scripts

  • Graph View
  • Dialogue Node
  • Action Node
  • Save Utility

Challenges

  • Navigating Graph Elements - Since there were a lot of different connections going on for each node and element, storing the information created a lot of complexity.

    • Solutions

  • Navigating Graph Elements - Implemented a solution using runtime scriptable objects for each element, storing their properties with identifiers for proper serialization and deserialization.


    • Animation Previewer ━ Unity / C#

    A tool made to tackle the challenge of previewing animations in relation to an object's hitbox.


    Features Implemented

    • Animation Detection - Scan and identifiy an object's animations in real-time
    • Animation Dropdown - List animtions in GUI
    • Continuous Playback - Play animations even when switching between different sequences
    • Playback Controls - View at different speeds to analyze animation relative to hitbox
    • Scrubbing through Animation - Scrub through the frames when animation is paused
    • Hitbox Resize Gizmo - Resize hitboxes to match animation
    • Rootmotion Application - Sync the player's local position with the animation's transformation, to line up animation with any placed hitboxes

    Scripts

    • Animation Preview

    Challenge

    One of the challenges I had was figuring out how to get an object's animations when it's selected in the hierarchy.

    Solution

    1. Animator Detection - Incorporated a check to see if the selected object had an animator component.
    2. Animation Clip Retrieval - Once the animator component was confirmed, implement a solution that accessed the animation clips by retrieving the runtimeAnimatorController associated with the object.
    3. Animation Clip Retrieval - If there were animations, the dropdown was set to play the first index. The remaining animations were dynamically populated in the dropdown.


    Image Canvas Resizer ━ WPF / C#

    I made this tool to optimize our project's overall footprint, by adjusting image dimensions without changing its quality. A tool for designers when making assets for our project.


    Features Implemented

    • Drag and drop - Load in a single image or an entire folder of images
    • File Path Label - See where the images are located in the file explorer
    • Resizing Options - Toggle between power of two or multiple of four dimensions, allowing customization based on specific requirements
    • Dimension Comparison Display - Display original and updated dimensions of the images (Single image only)
    • Popup Windows - Notify the user when the process is finished
